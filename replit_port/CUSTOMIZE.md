# コメントボックスの位置・大きさの変更方法

Webアプリ版（Replit版）で、コメント欄の位置や大きさを変更するには、**「画面上の見た目」**と**「PDFの出力設定」**の2箇所を修正する必要があります。

## 1. 画面上の見た目 (HTML/CSS)
ファイル: `replit_port/templates/index.html`

このファイルの `<style>` タグ内にある `#comment` という部分を探してください（70行目付近）。

```css
        #comment {
            left: 70px;   /* 左からの位置 */
            top: 56px;    /* 上からの位置 */
            width: 55px;  /* 横幅 */
            height: 500px;/* 高さ */
            font-size: 10px; /* 文字サイズ */
        }
```
ここの数値を変更すると、ブラウザ上の入力ボックスの位置や大きさが変わります。

## 2. PDFの出力設定 (Python)
ファイル: `replit_port/app.py`

このファイルの `COORDINATES` という設定を探してください（32行目付近）。

```python
    'コメント': {
        'x': [70, 80, 85, 95, 105], # 横位置（複数あるのは改行時の位置候補）
        'y': 786,                   # 縦位置（PDFの下からの位置）
        'font_size': 10,            # 文字サイズ
        'max_chars': 55,            # 1行の最大文字数
        'line_spacing': 20,         # 行間
        # ...
    },
```
- **位置を変えたい場合**: `'x'` の数字を変更します。
- **文字の大きさを変えたい場合**: `'font_size'` を変更します。
- **1行の長さを変えたい場合**: `'max_chars'` を変更します。

※ PDFの座標系は「左下が (0,0)」なので、`y` の値が大きいほど上に行きます。HTMLとは逆なので注意してください。
