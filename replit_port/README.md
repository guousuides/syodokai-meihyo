# 名票作成ツール (Web版 / Replit版) の使い方

このフォルダには、ReplitなどのWeb環境で動作する名票作成ツールが含まれています。

## フォルダ構成
- `app.py`: Webサーバーのプログラム (Flask)
- `templates/index.html`: 画面のレイアウトファイル
- `static/background.png`: 背景画像
- `generate_bg.py`: 背景画像を生成するスクリプト (通常は実行不要)

## 1. Replitで動かす場合

1.  **Replitでプロジェクトを作成**
    -   **こちらのリンクをクリックしてください: [https://replit.com/new](https://replit.com/new)**
    -   Templateに **"Flask"** を選択して作成します。

2.  **ファイルのアップロード**
    -   このフォルダ (`replit_port`) の中身 (`app.py`, `templates`, `static`) をすべてReplitにアップロードします。
    -   ドラッグ＆ドロップでアップロードできます。

3.  **フォントのアップロード (重要)**
    -   Windowsのフォントファイル `msmincho.ttc` (MS明朝) をReplitのファイル一覧の一番上（ルートディレクトリ）にアップロードしてください。
    -   ※これがないとPDF生成時にエラーになります。

4.  **実行**
    -   Replit画面上部の "Run" ボタンを押します。
    -   Webビューが表示され、入力フォームが出てきます。
    -   文字を入力して「PDF生成」ボタンを押すと、PDFがダウンロードされます。

## 2. 自分のパソコン (ローカル) で動かす場合

コマンドプロンプトで以下の手順を実行してください。

1.  **フォルダに移動**
    ```bash
    cd replit_port
    ```

2.  **ライブラリのインストール** (初回のみ)
    ```bash
    pip install flask reportlab pandas pillow
    ```

3.  **アプリの起動**
    ```bash
    python app.py
    ```

4.  **ブラウザで開く**
    -   起動すると `Running on http://127.0.0.1:8080` のような表示が出ます。
    -   ブラウザでそのURLを開いてください。

## 注意点
-   **縦書き入力**: ブラウザ上でも縦書きで入力できるようにしていますが、ブラウザの種類によっては表示が少しずれることがあります。
-   **PDFの見た目**: ブラウザ上のプレビューはあくまで簡易的なものです。最終的なレイアウトは生成されたPDFを確認してください。

## 3. みんなに共有する方法 (Replit)

このツールを他の人と共有して使ってもらうには、Replitで動かした状態で、**WebサイトのURL**を教えるのが一番簡単です。

1.  Replitで "Run" ボタンを押してアプリを起動します。
2.  画面右側（または別ウィンドウ）に表示される「Webview」のアドレスバーにあるURLをコピーします。
    -   例: `https://your-project-name. your-username.repl.co`
3.  このURLをみんなに送れば、スマホやPCからアクセスして使ってもらえます！

**注意点:**
-   Replitの無料プランでは、しばらく誰も使っていないとアプリが停止（スリープ）します。その場合、誰かがアクセスすると起動に少し時間がかかります（10〜20秒程度）。
-   完全に停止してしまった場合は、Replitの画面を開いてもう一度 "Run" を押す必要があります。
